# 友谊赛实战分析5维度

**Why:** 赔率模型按正式赛算友谊赛，但球员按友谊赛踢。6月4-5日体彩8场友谊赛，平局率62.5%，强队主胜仅12.5%，赔率<1.40的强队全部翻车。用户实战分析指出赔率完全忽略的关键因素。

**How to apply:** 友谊赛预测时，先评估5个维度再修正赔率隐含概率。赔率<1.40的强队主胜概率需大幅下调（-30~65pp），平局概率大幅上调（+10~20pp）。

## 5大维度

### 1. 雇佣关系：谁花钱请谁？
- 刚果花钱请丹麦当陪练 → 丹麦是"打工方"，不会拼命赢雇主 → 0-0
- **但**东道主雇主会认真打：墨西哥花钱请塞尔维亚，但墨西哥是WC东道主+高原主场 → 必须认真打
- 非东道主雇主也可能不认真：雇主只是想"试阵"，不一定要赢

### 2. 球迷效应：客场球迷数量
- 荷兰主场分7000张客场票给阿尔及利亚 → 主场优势被大幅削弱
- 跨境观赛=旅游动机 → 球迷超开心 → 球员士气加成
- 赔率1.18以为纯主场，实际是"客场主场"

### 3. 动机需求：球队当前需要什么？
- 连败需要信心 → 花钱请弱队找自信（瑞典预选赛6场2平4负，被科索沃双杀，请希腊找信心）
- 东道主需要展示 → 必须认真打（墨西哥）
- 已出线/无所谓 → 不认真打
- **关键：同样花钱请陪练，雇主动机不同，结果完全不同**

### 4. 疲劳临界：赛季负荷差异
- 欧冠决赛球员 vs 非欧冠球员 → 40-50场 vs 30场
- 世界杯前6天 = 疲劳临界点
- 关键球星缺阵（姆巴佩刚打完欧冠决赛）→ 攻击力折损远超排名差反映
- 德尚目标是调试战术+调整身体，不是赢球

### 5. 场地特殊：高原/气候/时差
- 墨西哥海拔2000m+ → 欧洲球队极不适应
- 客队度假心理 + 高原反应 → 战力再打折扣

## 赔率偏差量化

| 赔率区间 | 隐含主胜 | 实际主胜 | 偏差 |
|----------|----------|----------|------|
| <1.20 | ~80% | ~15% | 高估65pp |
| 1.20-1.40 | ~65% | ~25% | 高估40pp |
| 1.40-1.80 | ~50% | ~35% | 高估15pp |
| >1.80 | ~40% | ~35% | 基本准确 |

## 与[[model_accuracy_improvements]]的关系
友谊赛规则应作为场景增强的独立分支，不能套用联赛模型。现有模型把友谊赛当"弱化版正式赛"，实际是"完全不同的比赛"。
