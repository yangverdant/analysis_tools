## Oddsfe 229K场赔率基线回测

### 总体
- 229,906场FINISHED比赛, Pinnacle赔率argmax准确率: **54.14%**
- 实际分布: 主胜44.3% 平局23.5% 客胜32.2%

### 按赔率区间
| 区间 | 场次 | 准确率 | 平局率 |
|------|------|--------|--------|
| <1.30 | 16,973 | 82.88% | 10.7% |
| 1.30-1.60 | 29,722 | 67.00% | 18.8% |
| 1.60-2.00 | 46,435 | 53.41% | 25.0% |
| 2.00-3.00 | 79,140 | **41.38%** | 27.6% |
| >3.00 | 57,636 | 57.14% | 23.0% |

### 关键洞察: 2.00-3.00是模型最大价值区
- 赔率argmax只有41.38%, 因为赔率几乎不预测平局(只有2.8%场次argmax=draw)
- 但实际平局率27.6%, 模型如果能正确识别30%的平局 → 准确率可提升到44.69%(+3.3pp)
- 即使只识别20%也能提升到43.58%(+2.2pp)

### 平局阈值策略测试
在2.00-3.00区间, 简单draw_threshold策略反而降低准确率:
- threshold=0.25: 31.02% (低于41.37%基线)
- 原因: 预测draw时确实抓到一些平局, 但错失更多主胜

**结论**: 不能简单用阈值预测平局, 必须用模型判断。模型的优势在于综合多个因子(Elo+form+动机)来判断"这场是否可能平局"。
