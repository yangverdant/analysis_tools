## 二、关键缺失数据

### 1. ❌ 进球时间分布 - team_id缺失

**问题**: statsbomb_shots表的team_id字段全部为NULL

| 指标 | 数值 |
|------|------|
| 总射门记录 | 14,849 条 |
| 进球记录 | 1,734 条 |
| 有team_id的射门 | 0 条 (0%) |

**影响**: 无法按球队统计进球时间分布，分析中心显示 `[0,0,0,0,0,0]`

**解决方案**:
1. 从StatsBomb JSON数据重新填充team_id
2. 或使用API-Football的goalscorer端点获取进球时间
